# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Runix, please report it privately.

**Do not open a public GitHub issue.**

Instead, use one of the following methods:

- **GitHub Security Advisories**: [Report a vulnerability](https://github.com/RunixIO/runix/security/advisories/new)
- **Email**: Send details to the maintainers listed in the repository

Please include:

- A description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

We will acknowledge your report within 48 hours and aim to provide a fix or mitigation within 7 days for critical issues.

## Supported Versions

| Version | Supported           |
| ------- | ------------------- |
| 0.x     | Security fixes only |

## Disclosure Policy

We follow coordinated disclosure. Vulnerabilities will be disclosed after a fix is released and users have had reasonable time to update.
