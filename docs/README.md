# Runix Documentation

Welcome to the Runix documentation. Runix is a modern process manager and application supervisor written in Go, supporting Go, Python, Node.js/TypeScript, Bun, Deno, Ruby, and PHP through a unified CLI, TUI, Web UI, and MCP interface.
